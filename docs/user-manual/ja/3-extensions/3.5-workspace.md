# 3.5 Workspace ファイルと Daily Memory

## 概要

Workspace パネルは現在の版に組み込まれた汎用ワークスペースツールです。

YKW Bridge 自身の workspace ファイルと、日付ごとに整理された daily memory 記録を管理します。

デフォルトパス:

- `~/.ykw-bridge/workspace/`
- `~/.ykw-bridge/workspace/memory/`

アプリ設定ディレクトリを変更した場合、Workspace もその新しい設定ルートへ移動します。

## Workspace ファイル

現在編集できるファイル:

- `AGENTS.md`
- `SOUL.md`
- `USER.md`
- `IDENTITY.md`
- `TOOLS.md`
- `MEMORY.md`
- `HEARTBEAT.md`
- `BOOTSTRAP.md`
- `BOOT.md`

任意のファイルカードをクリックすると Markdown エディタが開きます。初回保存時にファイルは自動作成されます。

## Daily Memory

Daily Memory ファイルは次の形式で保存されます:

- `workspace/memory/YYYY-MM-DD.md`

できること:

- 今日の記録を作成
- すべての過去メモを検索
- 既存ファイルを編集
- 特定日の記録を削除

## 検索

Daily Memory は全文検索に対応しています:

1. 検索ボタンを押す、または `Cmd/Ctrl + F`
2. キーワードを入力
3. ヒット箇所とヒット数を確認

## ディレクトリを開く

パネル上部から次を直接開けます:

- `workspace/`
- `workspace/memory/`

システムのファイルマネージャーで直接確認したり、バックアップしたりするのに便利です。

## 注意事項

- これらのファイルは YKW Bridge の workspace に属し、Claude の公式設定ディレクトリへ自動書き込みはされません
- 編集できる workspace ファイル名はホワイトリスト方式です
- Daily Memory のファイル名は `YYYY-MM-DD.md` 形式である必要があります
