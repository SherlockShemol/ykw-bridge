# 3.5 Workspace Files and Daily Memory

## Overview

The Workspace panel is a built-in general workspace tool in the current edition.

It manages YKW Bridge's own workspace files and daily memory records organized by date.

Default paths:

- `~/.ykw-bridge/workspace/`
- `~/.ykw-bridge/workspace/memory/`

If you change the app config directory, Workspace moves with that new config root.

## Workspace Files

The currently editable files are:

- `AGENTS.md`
- `SOUL.md`
- `USER.md`
- `IDENTITY.md`
- `TOOLS.md`
- `MEMORY.md`
- `HEARTBEAT.md`
- `BOOTSTRAP.md`
- `BOOT.md`

Click any file card to open the Markdown editor. Files are created automatically on first save.

## Daily Memory

Daily memory files are stored as:

- `workspace/memory/YYYY-MM-DD.md`

You can:

- Create today's record
- Search all historical notes
- Edit existing files
- Delete records for a specific date

## Search

Daily memory supports full-text search:

1. Click the search button or use `Cmd/Ctrl + F`
2. Enter a keyword
3. Review match snippets and match counts

## Open Directories

At the top of the panel, you can open:

- `workspace/`
- `workspace/memory/`

This is useful for viewing or backing up files directly in the system file manager.

## Notes

- These files belong to the YKW Bridge workspace and are not automatically written into Claude's official config directory
- Only whitelisted workspace filenames can be edited
- Daily memory filenames must use the `YYYY-MM-DD.md` format
