# 3.3 Skills 管理

## 概览

Skills 是可复用的能力扩展。当前专版只保留 **Claude 工作流** 下的 Skills 管理。

YKW Bridge 使用统一的单一事实源（SSOT）来管理 Skills：

- 默认 SSOT：`~/.ykw-bridge/skills/`
- 备份目录：`~/.ykw-bridge/skill-backups/`

## 打开 Skills 页面

点击顶部导航栏的 **Skills** 按钮。

## 发现可安装 Skills

YKW Bridge 可以从预设仓库、你自己的 GitHub 仓库以及 `skills.sh` 公共目录中发现技能。

常见操作：

1. 搜索名称或描述
2. 按“已安装 / 未安装”筛选
3. 点击“刷新”重新扫描远端仓库

## 安装 Skills

1. 找到目标 Skill
2. 点击“安装”
3. 等待安装完成

安装后，Skill 会写入 SSOT，并同步到 Claude 工作流目录。

## 卸载与恢复

卸载前，YKW Bridge 会先创建备份。

卸载后会：

1. 从 SSOT 删除该 Skill
2. 从 Claude 工作流目录移除同步结果
3. 从数据库移除安装记录

如果需要恢复：

1. 打开 **从备份恢复**
2. 选择备份条目
3. 恢复后重新启用

## 仓库管理

你可以添加自定义仓库：

- Owner
- Repository
- Branch
- Subdirectory（可选）

删除仓库只会停止后续更新，不会自动删除已安装的 Skills。

## ZIP 安装与导入

除了在线仓库，还支持：

- 从 ZIP 文件安装
- 扫描本地未受管 Skills
- 导入现有 Claude 侧技能目录

## 存储位置切换

Skills 的源存储位置可以在两类目录之间切换：

| 位置 | 说明 |
| --- | --- |
| `~/.ykw-bridge/skills/` | 默认位置，由 YKW Bridge 管理 |
| `~/.agents/skills/` | 更适合多工具共用的社区约定目录 |

切换时不会丢失当前安装状态。

## 注意事项

- 当前专版只保留 Claude 侧 Skills 同步
- “同步方式”（软链接 / 文件复制）与“源存储位置切换”是两套不同设置
- 如果远程仓库不可访问，已安装 Skills 仍可继续使用
