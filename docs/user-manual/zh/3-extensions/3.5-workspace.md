# 3.5 工作区文件与每日记忆

## 概览

Workspace 面板是当前专版内置的通用工作区工具。

它用于管理 YKW Bridge 自己的工作区文件，以及按日期组织的每日记忆记录。

默认路径位于：

- `~/.ykw-bridge/workspace/`
- `~/.ykw-bridge/workspace/memory/`

如果你修改了应用配置目录，Workspace 也会跟着切到新的配置根目录。

## 工作区文件

当前允许编辑的文件包括：

- `AGENTS.md`
- `SOUL.md`
- `USER.md`
- `IDENTITY.md`
- `TOOLS.md`
- `MEMORY.md`
- `HEARTBEAT.md`
- `BOOTSTRAP.md`
- `BOOT.md`

点击任意文件卡片即可打开 Markdown 编辑器；首次保存时文件会自动创建。

## 每日记忆

每日记忆文件位于：

- `workspace/memory/YYYY-MM-DD.md`

你可以：

- 创建今天的记录
- 搜索全部历史记忆
- 编辑已有文件
- 删除指定日期的文件

## 搜索

每日记忆支持全文搜索：

1. 点击搜索按钮或使用 `Cmd/Ctrl + F`
2. 输入关键词
3. 查看命中片段与命中次数

## 打开目录

面板顶部可以直接打开：

- `workspace/`
- `workspace/memory/`

方便你在系统文件管理器里直接查看或备份内容。

## 注意事项

- 这些文件属于 YKW Bridge 工作区，不会自动写入 Claude 官方配置目录
- 只允许编辑白名单内的工作区文件名
- 每日记忆文件名必须是 `YYYY-MM-DD.md`
