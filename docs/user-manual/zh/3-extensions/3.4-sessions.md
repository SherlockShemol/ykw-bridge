# 3.4 会话管理器

## 概览

当前专版的会话管理器只浏览和管理 **Claude Code** 的本地会话日志。

会话文件来自 Claude 的项目会话目录：

- `~/.claude/projects/` 下的 JSONL 会话文件

其他旧工作流的会话入口已经从当前专版中移除。

## 打开会话管理器

点击顶部导航栏的 **Sessions** 按钮。

## 你可以做什么

- 浏览最近会话
- 按标题、目录、摘要或会话 ID 搜索
- 查看完整消息历史
- 复制恢复命令
- 删除单个或批量删除本地会话文件

## 界面结构

### 左侧列表

每条会话会显示：

- 标题
- 最后活跃时间
- 所属项目目录

### 右侧详情

选中后可以查看：

- 完整时间戳
- 项目目录
- 恢复命令
- 全部消息记录

## 搜索与筛选

当前筛选维度已经收敛为：

- **全部**
- **Claude**

按 `Esc` 可以快速清空搜索框。

## 恢复会话

点击 **恢复** 按钮后：

- 在 macOS 上，YKW Bridge 会尝试直接拉起终端
- 其他平台上，会优先复制恢复命令

如果终端拉起失败，也会回退为复制命令。

## 删除会话

### 删除单个会话

点击会话详情中的删除按钮，确认后会直接删除对应本地会话文件。

### 批量删除

1. 进入批量模式
2. 勾选多条会话
3. 点击批量删除

## 目录选择器

当前目录选择器主要针对 **Claude 会话恢复** 提供更稳妥的启动体验。旧版其他工作流的恢复路径已经不存在。

## 注意事项

- 会话按最后活跃时间倒序排列
- 只支持删除本地可写的 Claude 会话文件
- 没有本地源文件的只读记录不能删除
